📂 Upload Tips — Column Names

Name your Excel/CSV columns like this so the tracker can read them

✅ Core columns (required) Client   Expiration Date   P/ RN CX   Producer   Decription
☑️ Checkbox columns — use X, Yes, ✓, or 1 to mark done DocuSign Sent   Submitted   Quoted   Billed in Epic   Certs to Lender   Received Binder   Policy to Insured
🏷 Account-level columns Bound — sets account status   Notes — imported as a note entry
💡 Column names are not case-sensitive and partial matches work (e.g. "Sub" matches "Submitted"). After uploading, a summary will tell you which columns were detected. Imported checkbox data overwrites any previous value for that policy.
